Sistema de gestión de inventarios

Este proyecto consistió en desarrollar un sistema de gestión de inventarios para una empresa de manufactura. El sistema permitió a la empresa controlar y monitorear sus inventarios en tiempo real, lo que les permitió reducir los costos de almacenamiento y aumentar la eficiencia en su cadena de suministro.

  • Desarrollado en Python utilizando el framework Django
  • Integración con sistemas de contabilidad y facturación
  • Generación de informes y análisis de datos
